# Property Management Financial Operations & Compliance Dataset

This dataset package is designed for a property management analytics portfolio project.

It combines:
1. A real NYC DOF condominium comparable rental income property foundation.
2. Synthetic, connected operational tables for tenants, monthly financials, accounts receivable, vendor invoices, work orders, compliance reviews, and leasing pipeline.

The purpose is to support realistic SQL, Python, Power BI/Tableau, and responsible AI-assisted reporting workflows without forcing unrelated public datasets together.

Suggested project title:
**Property Management Financial Operations & Compliance Dashboard**

Recommended analysis:
- Budget variance and NOI/EBITDA reporting
- AR aging and collection priority
- Vendor invoice exception monitoring, including potential duplicate invoice review cases
- Work order SLA/backlog analysis
- Compliance review outcomes
- Property manager workload reporting
- Human-reviewed AI monthly reporting summary

Dashboard-ready exports:
- `data/processed/` contains SQL analysis outputs generated by `python src/run_sql_exports.py`.
- `data/dashboard_exports/` contains cleaned, business-friendly CSV tables generated by `python src/create_dashboard_exports.py`.
- The dashboard export files are intended for Tableau, Power BI, or similar BI tools and include executive monthly KPIs, property budget variance, AR tenant risk, invoice exceptions, work order backlog, compliance findings, and property manager workload tables.
